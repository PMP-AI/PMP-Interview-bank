## 作答說明

1. **作答環境：**  
   面試期間，電腦環境已經設定好，請直接在環境中編寫並執行 Python 代碼。

2. **版本控制：**  
   完成所有題目後，請自行建立一個新的 branch（例如：`interview-branch-您的姓名`），並將您的答案 commit 並 push 到該 branch，以便後續審查。
